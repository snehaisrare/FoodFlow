# FoodFlow – Food Ordering Platform

## 📌 Overview
FoodFlow is a full-stack food ordering platform inspired by real-world applications. It allows users to browse restaurants, explore menus, add items to cart, and place orders seamlessly.

---

## 🚀 Features
- User authentication (Login/Signup)
- Browse restaurants and menus
- Add to cart and place orders
- Order history tracking
- Search functionality for restaurants
- Ratings and reviews system
- Admin dashboard for managing users and orders

---

## 🛠️ Tech Stack
- Frontend: React.js, Redux
- Backend: Node.js, Express.js
- Database: MongoDB
