# FoodFlow – Food Ordering Platform

## 📌 Overview
FoodFlow is a full-stack food ordering platform inspired by real-world food delivery applications. It enables users to browse restaurants, explore menus, and place orders seamlessly while focusing on intuitive user experience and system functionality.

---

## 🚀 Features
- User authentication (Login/Signup)
- Browse restaurants and menus
- Add to cart and place orders
- Order history tracking
- Search functionality for restaurants
- Ratings and reviews system
- Admin dashboard for managing users and orders

---

## 🛠️ Tech Stack
- Frontend: React.js, Redux  
- Backend: Node.js, Express.js  
- Database: MongoDB
